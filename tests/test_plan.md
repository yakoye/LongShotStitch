# LongShotStitch v1.10 人工测试清单

## 粗细快捷档

- 几何图形、铅笔、荧光笔、直线、箭头、文字描边、马赛克等普通工具的粗细快捷按钮只显示 1 / 2 / 4 / 8。
- 普通工具界面不显示 16 / 32 快捷按钮。
- 普通工具数字输入框仍可以手动输入 16 或 32 并生效。
- 橡皮擦大小保留 16 / 32 快捷按钮。
- 鼠标悬停粗细区域滚轮向上变粗、向下变细。

## 二次点击编辑

- 选择文字工具，点击空白处新增文字。
- 选择文字工具，点击已有文字时选中并编辑，不新增第二个文字。
- 选择水印工具，第一次创建水印。
- 已存在水印时，再次点击水印工具或再次在画布使用水印，应编辑已有水印，不新增多层水印。
- 几何图形工具点击已有矩形 / 圆形时编辑，不新增。
- 箭头工具点击已有线条 / 箭头时编辑，不新增。
- 标号工具点击已有标号时编辑，不新增且不自动递增。
- 马赛克工具点击已有马赛克区域时编辑，不新增叠加。
- 铅笔 / 荧光笔点击已有笔迹时编辑，不新画一笔。
- 点击空白区域时，当前工具继续创建新对象，并保持当前工具不自动回到选择。

## 回归检查

- 工具区仍保持极简，不显示二级分类标题。
- 首次打开 / 清空后，中间区域仍可点击打开图片或工程。
- 分割线、拼接线、整体裁剪、单图裁剪基本功能不受影响。
- 导出 PNG 包含工具标注。
