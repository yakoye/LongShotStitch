# LongShotStitch v1.9 人工测试清单

## 工具属性极简化

- 进入工具模式，左侧工具区不显示说明卡片。
- 点击几何图形，只显示矩形、圆形、填充、颜色、填充色、线宽、透明度等可操作控件，不显示“形状 / 颜色 / 线条”等分类标题。
- 点击铅笔，只显示自由笔、荧光笔、颜色、线宽，不显示“铅笔 / 类型 / 样式 / 粗细”等标题。
- 点击箭头，只显示直线、单箭头、双箭头、颜色、线宽。
- 点击文字，只显示输入框、字号、颜色、描边、背景。
- 点击标号，只显示 1~10、颜色、大小。
- 点击马赛克，只显示矩形马赛克、自由马赛克、块大小。
- 点击橡皮擦，只显示大小控制。
- 点击水印，只显示文字、大小、颜色、透明度、密度 / 次数。

## 粗细 / 大小控制

- 点击不同粗细线条按钮，当前工具粗细立即改变。
- 鼠标悬停在粗细区域，向上滚变粗，向下滚变细。
- 在数字框输入数值，当前工具粗细精确改变。
- 选中已有圆圈、矩形、箭头、笔迹后，粗细按钮、滚轮、数字输入都能修改该对象。

## 回归检查

- 铅笔连续绘制后不会自动回到选择/移动。
- 圆圈、箭头、文字、标号连续使用后不会自动回到选择/移动。
- 首次打开 / 清空后，中间区域仍可点击打开图片或工程。
- 分割线、拼接线、整体裁剪、单图裁剪基本功能不受影响。
- 导出 PNG 包含工具标注。
