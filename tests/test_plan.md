# LongShotStitch v1.8 人工测试清单

## 基础打开

- [ ] 首次打开页面时，中间空白区域显示“打开图片 / 打开工程”。
- [ ] 点击“打开图片”可以选择图片。
- [ ] 点击“打开工程”可以选择 JSON 工程。
- [ ] 拖入图片可以打开。
- [ ] 拖入单个 JSON 工程文件可以打开工程。
- [ ] 点击清空后回到可点击的空白打开界面。

## 桌面端工具模式

- [ ] 工具模式左侧不出现说明卡片，只显示工具图标和必要属性。

- [ ] 进入“工具”模式后，工具图标显示在左侧“工具”下面。
- [ ] 工具图标只显示图标，不显示文字名称。
- [ ] 鼠标悬停在工具图标上能看到提示。
- [ ] 属性区域支持展开和隐藏。
- [ ] 切换几何、铅笔、箭头、文字、标号、马赛克、橡皮擦、水印正常。

## 手机端工具模式

- [ ] 手机窄屏下工具栏仍在底部附近，不跑到左侧。
- [ ] 手机窄屏属性浮层不会超出可视范围。

## 工具连续使用

- [ ] 选择铅笔后，画完一笔仍然保持铅笔。
- [ ] 选择圆形后，画完一个圆形仍然保持圆形。
- [ ] 选择箭头后，画完一个箭头仍然保持箭头。
- [ ] 选择文字后，插入文字仍然保持文字工具。
- [ ] 选择标号后，插入标号仍然保持标号工具，并按规则递增。
- [ ] 只有点击选择/移动或按 Esc 才回到选择/移动。

## 不回归检查

- [ ] 整体裁剪四边箭头和拖动逻辑正常。
- [ ] 单图裁剪外边界箭头和交界处拼接线逻辑正常。
- [ ] 分割线在可视范围内不乱跳，不在可视范围才回到屏幕中心。
- [ ] 拼接线编辑仍正常。
- [ ] 导出 PNG 包含标注内容。
