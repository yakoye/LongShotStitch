# v1.41 修改记录

## 工具交互

- 当前工具与当前选中对象彻底解耦。选中已有箭头、文字或形状不会自动切换工具。
- 方框、圆形、直线、箭头和矩形马赛克改为拖动创建；单击画面不再生成默认大小对象。
- 铅笔、荧光笔、自由马赛克增加有效轨迹阈值，轻触不会留下点或默认对象。
- 橡皮改为按住拖动擦除碰到的标注，单击不删除。
- 箭头、直线取消虚线包围框，只保留起点和终点控制柄；自由路径只显示端点提示。
- 标号工具图标统一为 `①`。

## 像素选区

- 新增“选取”工具，可拖出矩形像素选区。
- 支持按钮和 `Ctrl/Cmd+C`、`Ctrl/Cmd+X`、`Ctrl/Cmd+V` 完成复制、剪切、粘贴。
- 剪切后的源区域变为透明，粘贴内容成为可移动、缩放和删除的图像贴片。
- 贴片和透明挖空随撤销、重做、保存工程和打开工程保留。
- 超大选区增加像素上限保护，避免手机浏览器因 Canvas 内存过大崩溃。

## 删除与导出

- `Delete` / `Backspace` 删除当前选中标注，删除后不自动选择相邻对象。
- PNG、WebP、SVG 保留透明区域；JPEG 自动在白色背景上合成，避免透明区变黑。

## 测试

- 新增 `tests/tool_interaction_check.js`，固定工具锁定、拖动阈值、删除语义、选区剪贴板和透明导出规则。
- 更新桌面端、手机端、工程保存和多格式导出的人工回归清单。
