# v1.43.0 修改记录

## 标注入口与版本规则

- 版本升级到 `v1.43.0`。
- 采用 `v1.<功能迭代>.<修复号>`：功能迭代增加中间位，同一版本修复增加末位。
- 左侧“工具”改为“标注”，“选择”改为“选择 / 变换”，“像素选取”收敛为“选区”。

## 三种选区与自动变换

- 选区增加矩形、椭圆和自由路径三种形状。
- 统一选区包围框、路径绘制和离屏 Canvas 裁剪逻辑。
- 选区完成后自动提取透明 PNG，并建立可编辑 patch。
- 同时建立相同形状的 cutout，保持原位时视觉不重复，移动后源位置透明挖空。
- 选区自动进入“选择 / 变换”，可移动、缩放、旋转、调整透明度、复制和删除。
- 异步提取期间阻止重复画布操作；若用户已切换标注或更换画布，则取消过期的选区结果。
- patch 与 cutout 继续沿用 annotations、历史记录和工程 JSON，不改变旧工程主结构。

## 粘贴为可编辑对象

- 标注模式已有底图时，系统剪贴板图片粘贴为独立 patch，而不是新增拼接输入图。
- 外部图片按当前视口中心放置，并限制在合成画布范围内。
- 非标注模式或空工程继续沿用原来的图片导入流程。
- 顶部粘贴按钮和系统粘贴事件共用同一分流规则。

## 新建标注自动选择

- 形状、线条、箭头、画笔、荧光笔、标号和马赛克完成后自动进入“选择 / 变换”。
- 文字保持原位输入，提交后再进入选择状态。
- 水印继续保持唯一对象。

## 规则、测试与文档

- 增加 v1.43.0 静态约束和标注交互检查。
- 更新 README、界面规范、测试计划和标注系统任务书。
- `index.html` 与 `LongShotStitch.html` 保持一致。
