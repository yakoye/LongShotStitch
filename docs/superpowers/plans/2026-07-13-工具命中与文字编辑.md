# 工具命中与文字编辑 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让粘贴图块只能由选择工具编辑，并修复文字创建、尺寸匹配和双击编辑。

**Architecture:** 在 `onPointerDown` 按当前工具而不是命中对象分派事件；粘贴完成显式切换到选择工具。文字测量、画布渲染和原位编辑器共用同一套排版指标，双击只在选择工具下命中 `annotationSelect`。

**Tech Stack:** 单文件 HTML/CSS/JavaScript、Canvas 2D、Node.js 静态回归检查。
