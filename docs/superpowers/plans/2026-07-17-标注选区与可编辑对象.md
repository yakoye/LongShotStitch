# 标注选区与可编辑对象 Implementation Plan

**Goal:** 发布 v1.43.0，完成标注入口改名、三种选区自动转浮动对象、系统图片粘贴为对象和新建标注自动进入选择/变换。

**Architecture:** 延续单文件 Canvas 架构，通过 `patch + cutout` 组合表达非破坏浮动选区；所有对象继续放在 annotations 数组中，沿用现有历史、工程保存和导出管线。
