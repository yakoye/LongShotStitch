# 整体裁剪分摊与拼接线菜单 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 整体裁剪完成后保留多图结构和拼接线能力，并统一手机端/电脑端拼接线附加功能入口与按钮尺寸。

**Architecture:** 将 `commitCanvasCrop()` 从“合成单图”改为“把全局裁剪框映射回每张图片的 crop 参数”。手机端和电脑端复用同一组拼接线动作按钮，字幕不再作为独立桌面模式入口，只作为拼接线附加功能。

**Tech Stack:** 单文件 HTML/CSS/JavaScript，现有 Node 静态检查，Playwright/Chrome 手动自动化验证。

---

### Task 1: 整体裁剪分摊到原图

**Files:**
- Modify: `LongShotStitch.html`
- Modify: `index.html`
- Test: `tests/drag_math_check.js`
- Test: `tests/test_plan.md`

- [ ] **Step 1: 写失败检查**

在 `tests/drag_math_check.js` 中要求存在 `applyCanvasCropToImages`，并禁止 `commitCanvasCrop` 使用 `state.images = [record]` 作为整体裁剪完成路径。

- [ ] **Step 2: 确认失败**

Run: `node tests\drag_math_check.js`
Expected: FAIL，提示整体裁剪不能合成单图。

- [ ] **Step 3: 实现分摊函数**

在 `LongShotStitch.html` 中新增 `applyCanvasCropToImages()`：
- 基于 `computeLayout(true)` 获取裁剪前每张图在长图坐标中的 `dx/dy/dw/dh/scale/sx/sy/sw/sh`。
- 用 `normalizedCanvasCrop()` 得到保留矩形。
- 每张图与保留矩形求交集。
- 无交集的图片删除。
- 有交集的图片把交集反算成原图 crop：`left/top/right/bottom`。
- 同步 `state.seams`，清空 `state.canvasCrop`，重置 `splitDraft/cutLines/selected`。

- [ ] **Step 4: 改造 commitCanvasCrop**

`commitCanvasCrop()` 不再创建 canvas、不再 `state.images = [record]`，改为调用 `applyCanvasCropToImages()`。如果没有实际裁剪，只退出编辑态或切换到目标模式。

- [ ] **Step 5: 浏览器验证**

用 Chrome 导入两张图，纵向整体裁掉顶部超过第一张高度，确认第一张删除、第二张仍能进入单图裁剪；横向整体裁掉左侧超过第一张宽度，确认第一张删除、拼接线仍能编辑剩余相邻图。

### Task 2: 拼接线菜单和按钮统一

**Files:**
- Modify: `LongShotStitch.html`
- Modify: `index.html`
- Test: `tests/static_check.js`
- Test: `tests/test_plan.md`

- [ ] **Step 1: 调整桌面模式分类**

桌面侧栏去掉独立 `字幕` 模式按钮。保留拼接类按钮：`拼接线 / 整体裁剪 / 单图裁剪 / 分割线`，裁切和标注保留现有入口。

- [ ] **Step 2: 拼接线属性区增加四个等宽按钮**

在 `seamProps()` 顶部放置 `手动拼接 / 自动拼接 / 自动字幕 / 手动字幕`。按钮使用同一 CSS 类，宽高一致；点击行为复用 `runStitchAction()`。

- [ ] **Step 3: 手机端拼接线按钮增加更多提示**

手机端 `拼接线` 按钮显示 `···` 或同等提示，表达“里面还有选项”。底部弹出的四个按钮一行横向显示，宽高一致，可横向滚动，可隐藏；隐藏不退出当前功能。

- [ ] **Step 4: 单图裁剪图标统一**

将电脑端和手机端 `单图裁剪` 图标从 `▢¹` 改成方框里带 `1` 的图标，保证同组按钮大小一致，图标视觉尺寸一致。

- [ ] **Step 5: 验证交互**

手机端点击 `拼接线···` 展开四个等宽按钮；点击 `手动字幕` 进入字幕选区，隐藏菜单后仍能拖动字幕区域。电脑端拼接线属性区能看到四个等宽按钮，且没有独立字幕模式按钮。
