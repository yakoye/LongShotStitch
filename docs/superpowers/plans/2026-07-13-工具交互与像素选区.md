# 工具交互与像素选区 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让标注工具保持锁定、仅在有效拖动后创建对象，并提供可撤销、可保存的像素选区复制剪切粘贴能力。

**Architecture:** 保持单文件应用结构，在现有 `state`、指针事件和标注渲染管线中增加独立选区状态与 `patch/cutout` 标注类型。当前工具与当前对象彻底解耦，所有新行为通过现有历史快照和工程 JSON 持久化。

**Tech Stack:** HTML、CSS、原生 JavaScript、Canvas 2D、Node.js 静态回归检查、Chrome/Playwright 浏览器验证。
