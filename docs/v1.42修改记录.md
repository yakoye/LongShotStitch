# v1.42 修改记录

## 工具命中

- 只有“选择”工具可以移动、缩放、旋转或调整已有标注和粘贴图块。
- 铅笔、形状、箭头、文字、标号、马赛克和水印不再拖走已有对象；在对象上操作时继续执行当前工具的创建逻辑。
- 橡皮保持独立擦除逻辑，不进入对象移动状态。
- 像素选区粘贴完成后自动切换到“选择”，最新粘贴内容保持选中，可以立即移动和缩放。

## 文字编辑

- 修复双击命中类型错误，选择工具双击已有文字可以重新编辑。
- 桌面端新增文字后立即打开原位编辑框，手机端继续使用独占输入框唤起输入法。
- 编辑框和 Canvas 文字共用字体、字号、行高、内边距和文字测量结果。
- 单行、多行和竖排文字输入时，编辑框会随内容实时调整大小。
- 新建文字默认选中“输入文字”，直接输入即可替换；清空后结束编辑会删除空对象。
- 修改已有文字时，第一次实际输入建立一次撤销记录；只打开编辑框不会污染历史。

## 健壮性与性能

- 移动、缩放、旋转和线段端点调整在真实改变后建立撤销记录，单击对象不增加空历史。
- 标注拖动被系统手势取消时恢复到拖动前状态，不留下半移动对象。
- 粘贴图块拖动改用轻量克隆，不再序列化整段 Base64 图片数据。
- 大贴片先进入可选择、可拖动状态，再异步完成图片解码和重绘。
- 文字失焦延后到当前点击结束后收尾，避免第一次点击工具按钮被编辑器重绘吞掉。
- 低缩放时编辑框继续使用 Canvas 的真实缩放尺寸，不再被最小像素值放大。
- 键盘删除、方向键移动和标注复制剪切严格限制在“选择”工具。
- 双击文字先校验当前索引和对象类型，过期命中不会选错对象。
- 竖排文字中的换行按从右到左的多列排版处理。

## 规则与测试

- 新规则明确覆盖旧版“绘制工具优先编辑已有对象”和“禁止文字原位编辑”的冲突条款。
- 扩展 `tests/tool_interaction_check.js`，固定工具命中优先级、粘贴自动选择和文字双击编辑规则。
- 更新桌面端、手机端和文字排版人工回归清单。
