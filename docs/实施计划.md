# LongShotStitch 功能增强 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 增加自动拼接、字幕拼接、图片切割、标签页图标，并形成固定测试规则。

**Architecture:** 继续沿用单文件静态 HTML 的结构，在现有状态对象、模式按钮、属性面板、画布渲染和导出逻辑上做增量扩展。测试以 Node 静态检查和浏览器端手工/自动验证结合。

**Tech Stack:** HTML、CSS、原生 JavaScript、Canvas、Node.js 静态检查、Cloudflare 静态部署。

### Task 3: 自动拼接

**Files:**
- Modify: `LongShotStitch.html`
- Modify: `index.html`

- [ ] 在状态对象中增加 `autoStitchEnabled`。

### Task 4: 字幕拼接

**Files:**
- Modify: `LongShotStitch.html`
- Modify: `index.html`

- [ ] 增加 `subtitle` 模式按钮。
- [ ] 增加 `subtitleRegion` 状态，包含 `heightRatio` 和 `offsetRatio`。

### Task 5: 图片切割

**Files:**
