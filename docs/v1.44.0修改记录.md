# v1.44.0 修改记录

## 版本与交互规则

- 版本升级为 `v1.44.0`。
- 缩放控制统一为：普通矩形对象和自由路径使用 8 个手柄；文字、标号、水印使用四角等比例缩放；直线和箭头使用两个端点。
- 四边手柄只改变单轴尺寸，四角手柄改变两个方向；按住 `Shift` 后四角缩放锁定对象开始拖动时的宽高比。

## Shift 绘制与移动约束

- 矩形、椭圆、矩形马赛克和矩形/椭圆选区在创建时按住 `Shift`，自动形成正方形或正圆。
- 直线、单箭头和双箭头在创建或拖动端点时按住 `Shift`，吸附到 45° 间隔方向。
- 铅笔、荧光笔和自由马赛克绘制中按住 `Shift`，从当前锚点拉出直线；松开后继续自由绘制。
- 选择对象后按住 `Shift` 拖动，锁定为水平或垂直移动。

## 对象变换与图层

- 矩形、椭圆、马赛克、贴图和选区对象显示四角与四边共 8 个缩放手柄。
- 铅笔、荧光笔和自由马赛克增加完整变换框，可以再次移动、从 8 个方向缩放并旋转；旋转后仍按真实路径命中。
- 文字、标号和水印通过四角等比例调整字号，避免横向或纵向拉扁。
- 新增置顶、上移一层、下移一层和置底。
- 新增图层快捷键：`Ctrl/Cmd + ]` 上移一层，`Ctrl/Cmd + [` 下移一层；加入 `Shift` 后分别置顶和置底。
- 新增 `Alt/Option + 拖动` 快速复制对象。

## 选区原位置填充

- 矩形、椭圆和自由选区支持设置原位置填充方式：透明、白色、黑色、自定义颜色。
- 自动转浮动对象和显式剪切均记录 `fillMode` 与 `fillColor`。
- 透明模式使用真实透明挖空；其它模式使用对应颜色填充。
- 填充结果纳入撤销、重做、工程保存、重新打开及各格式导出。

## 文字工具

- 移除新建文字和选中文字中的背景框设置。
- 新建文字固定为透明背景。
- 打开旧工程时即使存在旧 `bg` 字段，也不再绘制和导出文字背景框。
- 保留文字颜色、字号、描边、横排/竖排、旋转、原位编辑和再次编辑。

## 标注一致性检查

- 检查形状、线条、箭头、画笔、荧光笔、文字、标号、矩形/自由马赛克、水印、橡皮、贴图和三种选区的创建与二次编辑路径。
- 自由马赛克改为沿路径连续采样，Shift 直线不再只显示起点和终点两个马赛克块。
- 新增 `tests/annotation_maturity_check.js`，锁定 Shift、8 点缩放、图层、填充方式和无文字背景框规则。
- 新增 `tests/annotation_tool_matrix_check.js`，逐项检查桌面端与手机端标注入口、所有标注类型和二次编辑能力；手机端补齐水印入口。
