# 测试规则

以后发布前按本文检查。工具界面保持少文字、图标优先，测试时同时看功能和显示效果。

## 自动检查

每次提交前运行：

```bash
node tests/entry_check.js
node tests/static_check.js
node tests/smoke_check.js
```

必须全部通过。

## 发布入口

- 打开 `index.html` 应直接进入工具，不出现中转链接。
- 打开 `LongShotStitch.html` 应进入同一工具。
- 浏览器标签页应显示 `✂` 图标。

## 导入与拼接

- 空状态下可以打开图片、拖入图片、粘贴图片。
- 单图导入后可以预览、裁剪、导出。
- 多图导入后可以纵向拼接、横向拼接。
- 开启 `自动` 后导入多图，应自动判断纵向或横向。
- 自动拼接后，拼接线仍可手动调整。
- 删除、上移、下移图片后，拼接线数量应正确。

## 裁剪与分割

- 整体裁剪四边可拖动，可输入数值。
- 单图裁剪四边可拖动，可输入数值。
- 分割线可在长图中插入，位置可调整。
- 去白边按钮不会破坏图片顺序和已有标注。

## 工具模式

- 工具栏在桌面端显示在左侧面板内。
- 工具栏在手机端显示在底部，不遮挡底部模式栏。
- 文字、箭头、双箭头、直线、矩形、圆形、画笔、马赛克、编号、擦除、水印都能创建。
- 工具的颜色、线宽、字体大小、边框、填充、透明度等参数可调整。
- 文字标注可选中、移动、旋转。
- 文字标注可双击进入文本框编辑。
- 输入文字或数字时，键盘快捷键不应误触发模式切换。
- `Backspace` 和 `Delete` 不应直接删除文字标注，删除应走明确按钮。

## 字幕拼接

- 进入 `字幕` 模式后，应看到字幕检测矩形。
- `高` 可以调整字幕矩形高度。
- `位` 可以调整字幕矩形在整页中的垂直位置。
- 字幕在底部约 20% 时可正常框住。
- 字幕不贴底、需要多裁底部画面时，可通过 `高` 和 `位` 修正。
- 点击 `生成` 后，应得到一张字幕长图。
- 生成后的字幕长图可继续裁剪和导出。

## 图片切割

- 进入 `切割` 模式后，应看到当前图片的切割线。
- `横` 可把图按上下方向切成多张。
- `纵` 可把图按左右方向切成多张。
- `格` 可按行列平均切割。
- `3×3` 可快速生成九宫格预览。
- 点击 `导出` 后，应下载多张 PNG。
- 文件名必须按顺序编号，例如 `image_01.png`、`image_02.png`。
- 扫描书一张图两页时，纵向二分应能得到左右两页。
- 朋友圈九宫格时，`3×3` 应导出 9 张图。

## 桌面端

- 在 1440×900 视口下，顶部按钮、左侧模式、属性面板、画布都不重叠。
- 工具参数面板文字不溢出。
- 画布缩放、拖动、小地图、滚动条可用。
- 预览模式不显示编辑框和控制线。

## 手机端

- 在 390×844 视口下，顶部按钮和底部模式栏不重叠。
- 7 个底部模式入口都能显示，不应挤出屏幕。
- 工具浮层不遮挡底部模式栏。
- 字幕和切割模式的参数面板可操作。
- 手指拖动画布、缩放、选择工具时没有明显错位。

## 健壮性

- 没有图片时点击导出、字幕生成、切割导出，应只出现提示，不报错。
- 非图片文件不会破坏状态。
- 连续导入、清空、撤销、重做后仍可继续使用。
- 大图导出超过浏览器限制时应提示。
- 保存工程再打开后，图片、裁剪、标注、模式参数应恢复。
