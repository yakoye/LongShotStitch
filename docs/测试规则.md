# 测试规则

以后发布前按本文检查。标注界面保持少文字、图标优先，测试时同时看功能和显示效果。

## 自动检查

每次提交前运行：

```bash
node tests/entry_check.js
node tests/annotation_maturity_check.js
node tests/annotation_tool_matrix_check.js
node tests/canvas_editing_check.js
node tests/static_check.js
node tests/smoke_check.js
node tests/drag_math_check.js
node tests/tool_interaction_check.js
```

必须全部通过。

## 发布入口

- 打开 `index.html` 应直接进入工具，不出现中转链接。
- 打开 `LongShotStitch.html` 应进入同一工具。
- 浏览器标签页应显示 `✂` 图标。

## 导入与拼接

- 空状态下可以打开图片、拖入图片、粘贴图片。
- 单图导入后可以预览、裁剪、导出。
- 多图导入后可以纵向拼接、横向拼接。
- 开启 `自动` 后导入多图，应自动判断纵向或横向。
- 自动拼接后，拼接线仍可手动调整。
- 删除、上移、下移图片后，拼接线数量应正确。

## 裁剪与分割

- 整体裁剪四边可拖动，可输入数值。
- 单图裁剪四边可拖动，可输入数值。
- 分割线可在长图中插入，位置可调整。
- 去白边按钮不会破坏图片顺序和已有标注。

## 标注模式

- 标注栏在桌面端显示在左侧面板内。
- 标注栏在手机端显示在底部，不遮挡底部模式栏。
- 手机端第一次点击标注图标只切换标注类型，不自动弹属性面板。
- 手机端再次点击同一个标注图标，或选中已有标注，才显示属性面板。
- 手机端文件按钮包含打开图片、打开工程和保存工程；自动、方向、撤销、重做、适合窗口、信息和导出保持在顶部一排。
- 文字、箭头、双箭头、直线、矩形、圆形、画笔、马赛克、编号、擦除、水印都能创建。
- 绘制期间当前子类型保持锁定；铅笔、荧光笔、自由马赛克完成一笔后保持当前工具，其它单次对象完成后切换到“选择 / 变换”。
- 矩形、椭圆和自由选区完成后自动生成浮动 patch 与同形状 cutout，并立即进入“选择 / 变换”。
- 标注模式已有底图时，系统剪贴板图片粘贴为可再次编辑的 patch；空工程或其它模式仍导入为拼接图片。
- 只有“选择 / 变换”可以移动、缩放、旋转或编辑已有标注和粘贴图块。
- 铅笔、形状、箭头、文字、标号、马赛克和水印在已有对象上操作时不得拖走或缩放该对象。
- 水印工具可以调整全局水印参数，但不得借此移动或缩放其它标注和粘贴图块。
- 矩形、圆形、箭头、直线、矩形马赛克必须拖动后创建，单击画面不得生成默认对象。
- 铅笔、荧光笔、自由马赛克和橡皮必须按住拖动才生效。
- 单指绘制中切换为双指缩放或收到 `pointercancel` 时，不得提交半成品，也不得破坏 redo 历史。
- 选区支持矩形、椭圆和自由路径；完成后自动提取并形成可移动、缩放、旋转的图像对象。
- 工具的颜色、线宽、字体大小、边框、填充、透明度等参数可调整。
- Shift 约束必须一致：框形创建等宽高、线条吸附 45°、自由笔临时拉直、对象移动锁定单轴、四角缩放保持比例。
- 普通对象和自由路径显示 8 个缩放手柄；文字、标号、水印只使用四角等比例缩放。
- 图层置顶、上移、下移、置底及对应快捷键生效，重排后保持原对象选中。
- 选区原位置支持透明、白、黑和自定义颜色；工程恢复与导出结果一致。
- 文字工具不得显示或导出背景框，旧工程的文字背景字段也应忽略。
- 手机端箭头、线条、边框默认不能过细。
- 文字标注可选中、移动、旋转。
- 桌面端新增文字后应立即打开与文字边界匹配的原位编辑框。
- 选择工具双击文字标注可重新进入文本框编辑；其它工具双击不得编辑已有文字。
- 单行、多行、竖排、旋转和缩放状态下，文字编辑框与最终文字边界应保持一致。
- 修改已有文字、移动、缩放和旋转对象后可以撤销；仅点击或仅打开编辑框不得增加空历史。
- 选中对象拖动时发生 `pointercancel`，对象应恢复到拖动前位置和尺寸。
- 手机端新增文字后应进入独占输入框，尽量调出输入法。
- 输入文字或数字时，键盘快捷键不应误触发模式切换。
- `Backspace` 和 `Delete` 删除当前选中标注；删除后保持空选中，连续按键不得删除其他对象。
- 箭头和直线只显示起点、终点控制柄；自由路径显示 8 点变换框并可再次缩放。
- PNG、WebP、SVG 保留透明剪切区；JPEG 导出时自动铺白色背景。
- 文字编辑期间 Canvas 不得绘制同一文字对象，输入层与最终文字不能重影。
- 铅笔、荧光笔、自由马赛克画完一笔后保持当前工具，不自动显示选择框。
- 选择模式单击空白、重复点击选择按钮或按 `Esc` 能取消选择；空白拖动仍能平移。
- 橡皮默认局部擦除，也可切换为对象删除，局部擦除随工程保存。
- 扩图只增加四边背景和画布，不拉伸底图与标注。
- 删除并收缩遵守 C 规则：矩形单边贴边、整宽或整高条带才允许收缩；普通内部选区只清除/填充。
- 指定像素、完整适应、填满裁边、拉伸和毫米/DPI 预设作用于最终导出尺寸。

## 字幕拼接

- 进入 `字幕` 模式后，应看到字幕检测矩形。
- `高` 可以调整字幕矩形高度。
- `位` 可以调整字幕矩形在整页中的垂直位置。
- 字幕在底部约 20% 时可正常框住。
- 字幕不贴底、需要多裁底部画面时，可通过 `高` 和 `位` 修正。
- 点击 `生成` 后，应得到一张字幕长图。
- 生成后的字幕长图可继续裁剪和导出。

## 图片切割

- 进入 `切割` 模式后，应看到当前图片的切割线。
- `横` 可把图按上下方向切成多张。
- `纵` 可把图按左右方向切成多张。
- `格` 可按行列平均切割。
- `3×3` 可快速生成九宫格预览。
- 切割线默认按设置平均生成。
- 切割线可手动拖动微调。
- 点击 `完成` 后应切换到单图裁剪。
- 顶部导出按钮使用格式下拉选择输出格式；切割模式按当前格式下载多张文件，其他模式按当前格式导出一张文件。
- 版本号只在用户可见功能、行为修复或发布文件结构发生变化时更新；当前版本必须写在页面标题、品牌小字和 README 中。
- 自动化测试应以 `LongShotStitch.html` 为主文件，不再依赖 `LongShotStitch_v*.html` 版本归档。
- 大图性能回归检查：小图不进入代理图模式；大图编辑优先使用代理图，导出继续使用原图。
- 文件名必须按行列编号，例如 `image_01_01.png`、`image_01_02.png`。
- 扫描书一张图两页时，纵向二分应能得到左右两页。
- 朋友圈九宫格时，`3×3` 应导出 9 张图。

## 拼接线与分割线拖动

- 拖动上方图片区域，只影响上方图片，不影响下方图片。
- 拖动下方图片区域，只影响下方图片，不影响上方图片。
- 横向拼接时，拖动左侧图片区域只影响左侧图片。
- 横向拼接时，拖动右侧图片区域只影响右侧图片。
- 往下或往右拖动当前图片时，当前图片应朝拖动方向变化。

## v1.46.0 连续绘制、对象属性与导出面板

- 铅笔、荧光笔和自由马赛克在按住拖动期间不得出现蓝色虚线框、缩放点、旋转点或首尾点。
- 松开并形成有效轨迹后，显示首尾标记、8 点缩放和旋转控制，同时连续工具仍保持激活。
- 取消选择后重新单击轨迹，必须立即显示该对象的线宽、颜色、图层和对象操作属性。
- 依次选择矩形、圆形、线条、箭头、自由路径、文字、标号、马赛克、水印和贴图，属性面板不得偶发消失或显示上一对象内容。
- 调整图层顺序后继续修改属性，仍应作用于原对象，不得因数组索引变化而切换对象。
- 桌面导出浮层实际渲染宽度不得超过 360 px；输出宽高约 76 px，毫米输入约 72 px，DPI 约 82 px。
- 390 px 移动视口下，导出浮层不得越过左右边界。
- 自动检查运行 `node tests/annotation_selection_export_check.js`，并在可用浏览器环境中执行真实指针绘制和 DOM 尺寸检查。

## 桌面端

- 在 1440×900 视口下，顶部按钮、左侧模式、属性面板、画布都不重叠。
- 工具参数面板文字不溢出。
- 画布缩放、拖动、小地图、滚动条可用。
- 预览模式不显示编辑框和控制线。

## 手机端

- 在 390×844 视口下，顶部按钮和底部模式栏不重叠。
- 7 个底部模式入口都能显示，不应挤出屏幕。
- 电脑端可用但手机顶部隐藏的按钮，应能在 `⋯` 菜单中找到。
- 工具浮层不遮挡底部模式栏。
- 字幕和切割模式的参数面板可操作。
- 手指拖动画布、缩放、选择工具时没有明显错位。

## 健壮性

- 没有图片时点击导出、字幕生成、切割导出，应只出现提示，不报错。
- 非图片文件不会破坏状态。
- 连续导入、清空、撤销、重做后仍可继续使用。
- 大图导出超过浏览器限制时应提示。
- 保存工程再打开后，图片、裁剪、标注、模式参数应恢复。
