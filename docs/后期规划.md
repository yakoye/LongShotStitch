# 后期规划

## 近期

- 增加图层列表，集中管理标注、贴片和透明剪切区的顺序、显示与锁定。
- 为贴片增加旋转、翻转和固定比例缩放，便于重排复杂界面截图。
- 改进选区边缘吸附，在靠近图片边缘、控件边框和文字行时提供轻量吸附。
- 增加选区导出，可只导出当前框选内容。

## 中期

- 将橡皮升级为局部擦除路径，同时继续只处理标注层、不破坏原图。
- 增加套索选区和同色区域选取，满足不规则界面元素移动需求。
- 增加贴片羽化、边缘融合和背景色智能填补，减少移动区域后的明显空洞。
- 对大贴片使用预览代理图，编辑时保持流畅，导出时映射回原始像素。

## 健壮性

- 建立真实 Android Chrome、iOS Safari、桌面 Chrome 和 Edge 的固定回归设备矩阵。
- 增加多次复制粘贴、连续撤销重做、工程跨版本打开和超大选区内存压力测试。
- 为浏览器 Canvas 尺寸、内存和导出失败提供统一错误提示与恢复路径。
