# 功能设计

## 目标

在现有 `LongShotStitch` 静态工具中补齐三个实用能力：自动拼接、字幕拼接、图片切割，并补上标签页图标和固定测试规则。

## 自动拼接

- 在顶部工具区提供短标签开关 `自动`。
- 勾选后导入多张图片时自动判断拼接方向。
- 判断规则保持简单：图片宽度更接近时使用纵向拼接，图片高度更接近时使用横向拼接。
- 自动拼接后继续使用现有接缝检测能力尝试裁掉重叠区域。
- 用户仍可继续手动改方向、裁剪、调整接缝和导出。

## 字幕拼接

- 新增独立模式 `字幕`。
- 默认从每张图中提取底部区域，适配电影、视频截图字幕。
- 提供短参数：`高` 和 `位`，分别控制字幕矩形高度和垂直位置。
- 画布上显示字幕检测矩形，便于确认字幕是否被框住。
- 点击生成后，将各图字幕区域按顺序拼成一张新图，进入正常编辑和导出流程。
- 不做 OCR，不识别文字，只处理图片区域。

## 图片切割

- 新增独立模式 `切割`。
- 支持选择单张图片作为切割对象。
- 支持按行、按列、按网格切割。
- 提供 `3x3` 九宫格预设。
- 画布上显示平均生成的切割线，用户可拖动微调。
- 点击 `完成` 后切换到单图裁剪。
- 顶部 `导出 PNG` 在切割模式下下载多张 PNG。
- 导出时按行列编号：`原文件名_01_01.png`、`原文件名_01_02.png`。

## 界面原则

- 工具里尽量少放说明文字。
- 优先用图标和短标签表达。
- 参数名称使用短字，例如 `高`、`位`、`行`、`列`、`导出`。
- 说明性内容放入文档，不占主界面空间。

## 不做

- 暂不开发 `FUTURE_FEATURES.md` 第三项性能软提示。
- 暂不做字幕 OCR。
- 暂不把切割导出打包成 ZIP。
