# LongShotStitch 标注系统任务书

## 1. 产品方向

LongShotStitch 不再把“工具”仅理解为若干一次性画笔，而是建立一个非破坏、可重复编辑的标注与图像对象系统：

- 原图、拼接结果保持为底图。
- 形状、文字、箭头、画笔、马赛克、水印、粘贴图片和选区提取内容都作为独立对象保存。
- 每个对象都能再次选中、移动、缩放；适合的对象还可旋转、改色、改线宽或继续编辑文字。
- 选区完成后立即进入“选择 / 变换”，不再停留在只能复制、剪切的临时框状态。

## 2. 版本规则

从本版开始使用三段式版本号：`v1.<功能迭代>.<修复号>`。

- `1`：当前主版本，维持现有单页网页架构和工程格式主线。
- 中间位：功能迭代，每次加入一组完整能力时递增，例如 `v1.43.0`、`v1.44.0`。
- 末位：同一功能版本上的缺陷修复，例如 `v1.43.1`、`v1.43.2`。
- 若未来重做工程数据结构、渲染架构或兼容方式，再升级到 `v2.0.0`。

## 3. v1.45.0 当前范围（P0）

### 3.1 文字与连续绘制

- 文字进入原位编辑时，Canvas 暂停绘制同一文字对象，避免输入层与最终文字叠加重影。
- 铅笔、荧光笔、自由马赛克属于连续工具；完成一笔后保持当前工具，不自动出现选择虚线框。
- 形状、箭头、矩形马赛克、选区与粘贴图片仍属于单次对象流程，完成后进入选择或变换。

### 3.2 取消选择与橡皮

- 选择模式单击画布空白、再次点击当前选择按钮或按 `Esc`，均可取消选中。
- 空白处发生实际拖动时按平移画布处理，不误判为取消选择。
- 橡皮默认使用局部擦除轨迹；“对象”模式保留整对象删除。
- 局部擦除数据随标注对象保存，参与撤销、重做、工程保存和重新打开。

### 3.3 扩图

- 画布支持上、下、左、右分别增加像素。
- 支持输入最终宽高、锁定比例与九宫格锚点。
- 扩图只增加画布背景，不拉伸底图、粘贴图片和标注。
- 扩展区域支持白、黑或自定义背景颜色，并保存到工程文件。

### 3.4 删除并收缩画布（C 规则）

- 只接受矩形选区。
- 贴住且只贴住一个边缘时，按该边缘的完整条带删除并收缩。
- 横跨全宽的矩形可删除横向条带；纵跨全高的矩形可删除纵向条带。
- 普通内部矩形、椭圆、自由选区以及同时贴住两个相邻边的角落选区，不允许二维收缩，只执行透明/白/黑/自定义颜色清除。
- 删除边缘条带时复用画布裁剪逻辑；删除内部整宽/整高条带时合并底图，但保留未跨带标注，跨带标注转为可编辑图片片段。
- 所有删除收缩操作均可撤销。

### 3.5 精确尺寸与证件照输出

- 最终导出支持指定宽度、高度或只输入一边并按原比例计算另一边。
- 尺寸比例不一致时支持完整适应、填满裁边和拉伸三种方式。
- 支持 128×128、256×256 图标预设。
- 常用 1 寸默认 25×35 mm、2 寸默认 35×49 mm，DPI 与毫米尺寸均可修改后重新计算像素。
- 裁剪比例增加 25:35、35:49，先通过裁剪框调整构图，再执行目标像素导出。
- 指定像素只改变最终文件，不改变工程内部对象坐标和底图分辨率。

### 3.6 v1.44.0 已完成能力继续保留

- Shift 绘制和缩放约束、8 点变换、直线端点调整。
- 图层置顶/置底和前后移动、Alt/Option 拖动复制。
- 选区原位置透明、白、黑、自定义颜色填充。
- 无文字背景框、自由路径再次编辑和旋转命中。

## 4. v1.43.0 已完成范围

### 3.1 名称与入口

- 左侧模式“工具”改名为“标注”。
- 选择工具改为“选择 / 变换”。
- 像素选取改为“选区”，提供矩形、椭圆、自由选区。

### 3.2 选区即对象

- 矩形、椭圆、自由选区拖动完成后，自动提取画面内容。
- 自动在原位置生成可编辑图像对象，并自动切换到“选择 / 变换”。
- 选区对象立即显示变换框，可移动、缩放、旋转、调整透明度、复制、删除。
- 提取时在底图位置增加同形状透明挖空，使移动后的行为接近 Windows 画图的浮动选区，而不是简单叠加一份副本。
- 选区对象及其透明挖空都纳入撤销、重做、工程保存和重新打开。

### 3.3 粘贴即对象

- 标注模式已有底图时，系统剪贴板图片粘贴为可编辑图像对象。
- 没有底图或不在标注模式时，粘贴仍作为拼接输入图片导入。
- 应用内复制、剪切、粘贴继续支持标注对象和选区对象。
- 所有粘贴对象保持独立，之后可以再次选中、移动、缩放、旋转和删除。

### 3.4 标注后自动变换

- 新建形状、箭头、线条、标号、画笔、荧光笔、马赛克完成后，自动切换到“选择 / 变换”，并保持新对象选中。
- 新建文字完成输入后进入选择状态；输入过程中不强制抢走焦点。
- 水印仍保持全局唯一对象规则。

## 5. Snipaste / PixPin 能力全量功能池

下面不是要求一次性塞进一个版本，而是建立完整产品边界，并按 LongShotStitch 的对象模型逐项实现。

### 4.1 截图与选区

- 手动矩形选区、椭圆选区、自由选区、全屏和预设区域。
- 窗口与 UI 元素自动识别、父子层级切换、历史选区、历史截图。
- 像素级方向键微调、十字辅助线、尺寸显示、放大镜、实时取色与多种颜色格式。
- 延时截图、显示/隐藏鼠标、HDR 与多显示器适配。
- 截图后复制、保存、快速保存、贴图、打印和自定义动作。

### 4.2 标注与二次编辑

- 矩形、圆角矩形、椭圆、折线、直线、单箭头、双箭头和更多箭头端点。
- 序号、文字、铅笔、荧光笔、聚光灯、放大镜和水印。
- 矩形/自由/椭圆马赛克、高斯模糊、自动马赛克、智能擦除和橡皮擦。
- 颜色、透明度、填充、描边、粗细、圆角、字号、横排/竖排、旋转等属性。
- 撤销/重做、删除、复制、二次编辑、中心缩放、等比例缩放和像素微调。
- 多选、分组、对齐、分布、锁定、隐藏和图层顺序。
- 工具栏主/次区域、自定义按钮顺序、各工具独立记忆颜色与粗细。

### 4.3 贴图与对象管理

- 图片、文本、HTML、颜色、文件/文件夹和剪贴板内容贴图。
- 贴图移动、缩放、旋转、水平/垂直翻转、灰度、反色、亮度和透明度。
- 裁剪、替换贴图、重复贴图策略、恢复最近关闭、启动时恢复。
- 置顶、鼠标穿透、锁定、隐藏/显示全部、缩略图模式和贴图边框缩放。
- 贴图历史、贴图分组、跨组移动、组颜色和不同工作场景切换。
- 二维码/条码识别、图片翻译、贴图文字选择与再次标注。

### 4.4 长截图与拼接

- 纵向、横向滚动截图，手动滚轮与自动滚动。
- 智能重叠检测、实时预览、暂停/继续、区域移动和区域尺寸调整。
- 超长图分块渲染、内存保护、错误区块删除、反向滚动自动裁剪。
- 当前 LongShotStitch 的多图拼接、接缝调整、字幕拼接、切割与裁剪继续保留并增强。

### 4.5 OCR、识别与翻译

- 本地 OCR、区域文字选择、多语言识别和识别历史。
- 表格识别并导出 Excel、LaTeX 公式识别。
- 二维码、Code128、EAN13、UPCA、Code39、DataMatrix、PDF417 等识别。
- 图片到文本翻译、图片到图片翻译和普通文本翻译。
- 自动识别敏感文字并批量马赛克。

### 4.6 屏幕录制与动图

- 区域录制、暂停/继续、延时开始、帧率与画质设置。
- 鼠标轨迹、点击效果、按键显示、录制中标注。
- 录制后裁剪、时间轴编辑和局部遮挡。
- MP4、GIF、WebP 导出；普通录制与快速录制两种流程。

### 4.7 保存、工程与自动化

- PNG、JPEG、WebP、SVG、PDF 保存，质量预览、分页和页边距。
- 工程文件保存/恢复、自动恢复、历史记录和异常崩溃保护。
- 快捷键、预设动作、自定义动作、脚本/命令行调用和忽略特定程序。
- 主题、语言、多屏、DPI、配置导入导出与跨设备同步。

### 4.8 实现载体分级

- **纯网页可可靠实现：** 当前拼接、裁剪、切割、标注对象、图片粘贴、长图算法、工程保存、网页内 OCR/翻译、导出。
- **浏览器扩展更合适：** 当前网页截图、整页截图、页面滚动控制、网页元素识别、网页贴图侧栏。
- **桌面壳才能完整实现：** 全局快捷键、任意窗口/UI 检测、跨应用截图、系统置顶贴图、鼠标穿透、多显示器全局贴图、桌面录屏、文件贴图和托盘动作。

## 7. 交互原则

- “选择 / 变换”工具点中已有对象：选择该对象并显示变换框。
- 绘制工具保持创建语义，即使从已有对象上开始拖动也创建新对象；连续工具保持当前工具，单次对象工具完成后进入选择状态。
- 选择工具单击空白：取消对象选择；按住拖动空白：平移画布。
- 选区工具完成后：自动创建浮动图像对象并进入选择状态。
- 所有公开界面保持图标化、短标签、紧凑，不展示开发说明。
- 桌面端：顶部工具栏 + 左侧模式栏 + 左侧属性面板 + 中间画布。
- 手机端：底部模式栏 + 紧贴底栏的标注工具栏 + 紧凑属性浮层。
